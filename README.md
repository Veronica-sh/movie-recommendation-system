Movie Recommendation System
This is a Python project that recommends movies based on user preferences such as genre, mood, and runtime.

Features:
Suggests movies based on user input
Uses a movie database stored in JSON
Saves user data for future recommendations

Files:
cinematch.py – main program
movies.json – movie database
user_data.json – user data storage

How to run:
Download or clone the repository
Make sure all files are in the same folder
Run the program using:
python cinematch.py
